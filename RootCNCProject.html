Root CNC 

The Root CNC family is an evolution of small and large Computerized Numerical Control (CNC) routers, with an emphasis on 3D printed parts and off-the-shelf hardware components that can be constructed in any home garage.  

Root 2 CNC:  Recommended build size between xx - xx and yy - yy.  Belt driven CNC with wooden base-board and side panels.  Replaced by Root 3 CNC.

Root 3 CNC:  Recommended build size between xx- xx and yy - yy.  Belt drive CNC with wooden base-board and side panels.  Nema 23 stepper motors.

Root 3 Lite:  Compact belt-driven version of Root 3 with smaller (Nema 17) stepper motors.  Flexible, convenient and powerful option where large cutting areas not required.  Further information available at the Root 3 Lite Github repository and website.

Root 4 CNC:  Recommended build size between xx- xx and yy - yy.  Heavy duty, ballscrew driven X, Y and Z axes, with Nema 23 stepper motors.  Designed to use AC Variable Frequency Drive (VFD) driven AC spindle, but flexible enough to use other spindle/router options.  Reference material available at the Root 4 CNC Github repository and website.

Root 4 Lite:   Compact version of Root 4 with smaller (Nema 17) stepper motors.  Convenience of a smaller, lighter footprint, lower cost high performance CNC capable of working wood, plastics and light metals.  Recommended build size working area between 174x230mm to 974x980mm.  Reference material available at the Root 4 Lite Github repository and website.  

Root Controller:  Flexible ESP32 GRBL (FluidNC) isolated motion controller designed to provide ultimate flexibility across a wide range of hardware and operating conditions.  Perfectly suited for larger CNC machines, but not limited to Root CNC.  Further information available at the Root Controller ISO Github.
